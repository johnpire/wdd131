Website Plan: WordCencus

Site Name

WordCencus - A text analysis tool designed to give users deep insights into their writing. The name plays on “word census,” representing the collection and analysis of textual data.

Reason for choice: It's concise, memorable, and clearly reflects the site's purpose.

Optional domain availability: wordcencus.com

Site Purpose

The purpose of WordCencus is to provide an interactive text analyzer that processes user input to deliver:

  • Word frequency statistics
  • Character usage analysis
  • Sentence structure insights
  • Visual data representations of textual patterns

It serves writers, students, and content creators by transforming raw text into actionable insights while showcasing practical JavaScript and data visualization skills.

Scenarios

  • “I'm writing an essay—how can I see which words I'm overusing?”
  • “Can I quickly check how many sentences I have and their average length?”

Color Schema

  • Primary Color: Drab Dark Brown (#504B3A) - Used for text, main headings, and border.
  • Secondary Color: Mindaro (#DDE392) - Used for main/section's background color.
  • Accent 1: Light Sea Green (#2CA6A4) - Used for section titles and highlights.
  • Accent 2: Persian Orange (#DE9151) - Used for subheadings and call-to-action elements.

Typography

  • Heading Font: Arial Black - For all headings to ensure clarity and emphasis.
  • Body Font: Arial, sans-serif - For all paragraph text for readability.
  • Special Highlight Font: Georgia - For quotes or special emphasis text.
